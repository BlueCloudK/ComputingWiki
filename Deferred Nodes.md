# Deferred Nodes

Các node này đã xuất hiện trong Maps/Report nhưng chưa tạo thành node chính vì chưa có source trace đủ rõ trong Maps hiện tại.

## Hardware / Low-Level

- PCI Express - hardware-specific; giữ lại để xét sau khi bổ sung source trace rõ.
- Magnetic Tape - storage media quá thấp; giữ lại để xét sau khi bổ sung source trace rõ.

## Machine Learning

- K Nearest Neighbors - chưa có AI/ML map đủ rõ trong batch hiện tại.
