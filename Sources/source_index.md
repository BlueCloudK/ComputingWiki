## Quy tắc dùng source

- Chỉ lấy tên node, node con, quan hệ cha-con, thuật ngữ chuẩn.
- Không lấy tutorial, ví dụ code dài, lịch sử, tranh luận học thuật.
- Không đưa framework/tool cụ thể vào core nếu chưa cần.

## Quy tắc ngôn ngữ

- Canonical node/file/link name: English.
- Aliases: English + Vietnamese nếu có tên Việt dễ hiểu.
- Readable explanation trong MOC/Nodes: tiếng Việt ngắn gọn.
- Source trace trong Maps: giữ English theo nguồn gốc.
- Maps có thể dùng English nhiều hơn vì là bản trích source; MOC/Nodes ưu tiên tiếng Việt để đọc nhanh.
