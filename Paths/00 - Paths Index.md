# Paths Index

Paths là các tuyến đi qua ComputingWiki theo thứ tự học, thiết kế hoặc làm project.

Khác với MOC, path không cố gom toàn bộ node trong một vùng. Path chỉ chọn các node cần đi qua để hiểu một workflow hoặc mảng thực hành cụ thể.

## Cách dùng

- Bắt đầu từ path gần với việc đang làm.
- Đi theo stage từ trên xuống dưới.
- Khi gặp node lạ, mở node đó rồi quay lại path.
- Nếu path thiếu khái niệm, ghi vào `Missing / Future nodes` dưới dạng text thường trước khi tạo node mới.
