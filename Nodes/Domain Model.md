# Domain Model

Aliases: domain model, mô hình miền nghiệp vụ

Domain Model là cách gọi tên và mô tả các đối tượng chính trong bài toán.

## Dùng để làm gì

Dùng để thống nhất ngôn ngữ giữa requirement, database, API và code.
