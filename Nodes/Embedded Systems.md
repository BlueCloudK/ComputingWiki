# Embedded Systems

Aliases: embedded system, hệ thống nhúng

## Dùng trong dự án để làm gì

Embedded Systems giúp hiểu phần cứng và tổ chức máy tính ảnh hưởng thế nào tới hiệu năng, I/O và tài nguyên. Nó hữu ích khi debug bottleneck thấp tầng hoặc cần biết giới hạn thực tế của hệ thống chạy trên máy.

## Khi nào cần quan tâm

- Bottleneck liên quan CPU, memory, cache hoặc I/O
- Cần hiểu giới hạn phần cứng khi tối ưu
- Debug vấn đề ở tầng máy hoặc embedded

## Lỗi / rủi ro thường gặp

- Tối ưu sai bottleneck vì không đo tầng thấp
- Không tính cache/I/O nên performance lệch dự đoán
- Phụ thuộc phần cứng làm khó deploy sang môi trường khác
